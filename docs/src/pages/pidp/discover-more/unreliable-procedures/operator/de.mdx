# Handhabung durch den Anbieter

## Bewertung der Handhabung beim Betreiber einer digitalen Plattform

### Allgemein

Große Unternehmen, welche mächtige Social-Media Plattformen betreiben, verfügen
über einen internen Richtlinienkatalog, der bestimmt, ob ein vom Nutzer
gemeldeter Inhalt in seiner Aussage das Kriterium erfüllt, um als kritisch
eingestuft zu werden.
