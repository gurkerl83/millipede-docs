# Notwendigkeit eines Perspektivenwechsels

Die Motivation, welche zur Erstellung der vorliegenden Ausarbeitung führt ist vielfältig.

Unter anderem dient die Ausarbeitung als Positionspapier, um einen Überblick oder eine Orientierung dieser, als äußerst komplex geltender Problemstellung zu erlauben.
Hierzu wird eine Zerlegung der Problemstellung in einfach zu verstehende Teilproblemstellungen ausgeführt.

Das Ziel besteht in der Sensibilisierung des Bewusstseins von Entscheidungsträgern, den Fokus auf relevante Themenbereiche zu richten, welche als Ursache der Problemstellung gelten.
