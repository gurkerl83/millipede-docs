# Method hooking

> Method hooking beschreibt ein Verfahren, bei dem eine Methode einer Anwendung
> zur Laufzeit mit einer anderen, neu erstellten Methode ersetzt wird. Die
> ursprüngliche Methode bleibt erhalten und kann aufgerufen werden, jedoch ist
> dies nicht erforderlich. Auf diese Art und Weise wird derselbe Methodenaufruf
> von der Anwendung durchgeführt; es wird jedoch eine andere Methode ausgeführt.
