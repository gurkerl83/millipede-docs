# Angriffsvekoren

## Warum eine neutralisierende Wirkung von (digitalen) Technologien bedingt wird, am Beispiel

In der IT-Sicherheit sind eine Reihe technischer Verfahren etabliert, welche eine Neutralisierung von schadhaften Software-basierten Anwendungen oder Komponenten etwa von Viren, Spyware, Malware und Ransomware erlauben.

Obwohl es eine oberflächliche Ähnlichkeit zwischen dieser und der in der vorliegenden Ausarbeitung formulierten Problemstellung gibt, muss festgehalten werden, dass eine andere Zielsetzung durch eine vom Angreifer bzw. Akteur ausgeführte Handlung verfolgt wird.

Um die Relevanz dieser Problemstellung zu begreifen muss erkannt werden, dass der Betrieb heutiger IT-Systeme und Infrastrukturen nicht ohne den Einsatz wesentlicher Komponenten zur IT-Sicherheit möglich ist. In der folgenden Tabelle wird die Ähnlichkeit der Problemstellungen hinsichtlich der vom Angreifer / Akteur verfolgten Zielsetzung, Auswirkung, Ursache und der eingesetzten Werkzeuge gegenübergestellt. Der Leser interpretiert die Tabelle von unten nach oben.
